# 密码生成器

安全的随机密码生成工具。

## 文件说明

- `password_gen.py`: 主程序文件

## 运行方法

```bash
cd password-generator
python password_gen.py
```

## 功能

- 生成自定义长度的随机密码
- 可选择包含的字符类型（大写、小写、数字、特殊字符）
- 密码强度检查
- 密码安全性评估和改进建议

## 使用示例

