# 待办事项管理器

简单的命令行待办事项管理程序。

## 文件说明

- `todo.py`: 主程序文件
- `todos.json`: 数据存储文件（运行时自动生成）

## 运行方法

```bash
cd todo-list
python todo.py
```

## 功能

- 添加新的待办事项
- 删除待办事项
- 标记待办事项为完成
- 查看所有待办事项
- 数据持久化存储

## 使用示例

```
请输入命令: add 学习Python
已添加待办事项: 学习Python

请输入命令: add 完成作业
已添加待办事项: 完成作业

请输入命令: list
=== 待办事项列表 ===
○ [1] 学习Python (2024-01-01 10:00:00)
○ [2] 完成作业 (2024-01-01 10:01:00)

请输入命令: complete 1
已完成待办事项: 学习Python
```
